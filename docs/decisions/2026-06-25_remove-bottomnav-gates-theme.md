# 하단바 제거 · 지도 입출구 UI · 다크토글 제거 · B602

**날짜**: 2026-06-25
**관련 파일**: map-view.tsx, exhibition-map.tsx, app-bar.tsx, (visitor)/page.tsx, floorplan-sibf.json, 3개 전시 하위 페이지

## What
- 하단 탭바를 없앴다. 지도는 하단에 검색+동선만 깔끔하게. "다른 사람 동선"은 상단바 아이콘으로 간다.
- 지도 하단 시트가 접히면 검색창만 보이고, 올리면 입구/출구 선택 → 분야 필터 → 부스 목록 순으로 나온다.
- 입구/출구를 지도에서 직접 고를 수 있다(고른 게이트로 동선이 그려진다).
- 라이트/다크 전환 버튼을 없앴다(시스템 설정을 따른다).
- 지도에서 빠져 있던 B602(문학동네·나무의마음)를 추가하고, 큰 단독 부스도 번호 대신 이름으로 표기한다.

## Why
- 하단 탭바가 사용자가 의도한 형태가 아니었고(특히 커뮤니티는 요청한 적 없음), 지도 하단을 어지럽혔다.
- 접힌 시트에 분야 칩이 잘려 같이 보였다 → 검색창만 깔끔히.
- 동선의 입구/출구를 정하는 UI가 통합 과정에서 사라졌다.
- 다크/라이트 토글이 불필요했다.
- B602 부스가 지도에 없었다.

## 판단 근거
- 지도를 단일 화면으로 유지하되, 진입은 상단바 아이콘으로 가볍게. 입출구는 "검색 아래·카테고리 위"라는 요청 위치에 배치.
- 큰 부스 이름 표기 기준을 facility 외에도 단변 ≥120px 단독 부스로 확장.
- B602는 부스 콘텐츠엔 있으나 도면 좌표만 없어, 겹치지 않는 위치에 추가(정확 좌표는 공식 도면 확보 시 보정).
