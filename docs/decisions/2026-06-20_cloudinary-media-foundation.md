# 커뮤니티 미디어(사진·짧은영상) 기반 — Cloudinary

**날짜**: 2026-06-20
**관련 파일**: env, cloudinary 설정, 업로드 서명 API

## What — 무엇이 달라졌나
(기반 단계) 실시간 커뮤니티에 사진·짧은 영상을 올릴 수 있도록 미디어 저장소(Cloudinary) 연결을 준비했어요. 아직 화면에 첨부 버튼은 없고, 업로드를 안전하게 처리할 토대만 마련된 상태입니다.

## Why — 왜 필요했나
방문객이 글로만 제보하다 보니 굿즈·대기 줄·부스 현장 같은 건 사진/영상으로 보여줘야 훨씬 잘 전달됩니다. 텍스트만으로는 한계가 있었습니다.

## 판단 근거
사진·영상은 "보여주는 정보"일 뿐 AI가 계속 학습할 대상은 아니라고 판단해, LLM에 넣지 않고 표시 전용으로 둡니다. 큰 파일이 서비스 서버를 거치지 않도록 브라우저에서 미디어 저장소로 바로 올리게 하고, 그 권한(서명)만 서버가 잠깐 내줍니다 — 비용·속도·보안 모두 유리. 자격증명은 코드에 넣지 않고 환경설정에만 둬 유출을 막습니다.
