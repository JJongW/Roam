# 지도 다듬기 + 대기시간 기능 제거 (배치 A)

**날짜**: 2026-06-24
**관련 파일**: exhibition-map.tsx, map-view.tsx, 그리고 대기(waiting) 관련 전 파일

## What — 무엇이 달라졌나

- 지도 위 동선(걷는 길)이 더 굵고 진하게 보이고, 살짝 그림자가 생겨 겹치는 구간이 층처럼 떠 보인다.
- 지도 확대/축소 조작이 한결 부드러워졌다. 부스를 눌러도 화면이 휙 끌려가지 않고, 이미 들여다보던 배율은 그대로 둔다.
- 부스 목록 검색창 오른쪽의 사진·둘러보기 버튼이 사라지고, 그 자리 대신 검색창 바로 아래에 분야(카테고리) 칩이 놓였다. 지도 위쪽 영역도 그만큼 깔끔해졌다.
- 지도에서 담은 동선을 한 번에 비우는 "동선 비우기" 버튼이 생겼다.
- 부스마다 표시되던 "대기 시간/혼잡도"가 앱 전체에서 사라졌다.

## Why — 왜 필요했나

- **동선 가시성**: 선이 너무 얇아 눈에 잘 안 들어왔고, 동선이 겹치는 곳에서 어느 길이 위인지 알 수 없었다.
- **줌 조작감**: 확대/축소가 부자연스럽다는 의견이 많았고, 부스를 누를 때마다 과하게 확대·이동돼 거슬렸다.
- **검색 옆 버튼**: 사진 판독·휙휙 둘러보기 버튼은 실제로 쓸 이유가 약했고, 정작 자주 쓰는 분야 필터가 검색 가까이에 없었다.
- **동선 비우기**: 한번 담은 동선이 계속 남아, 새로 시작하고 싶어도 비울 방법이 없었다.
- **대기 시간 제거**: 원래 판매자(부스 운영자)가 혼잡도를 올려 방문객에게 대기 예상치를 보여주려던 기능이었는데, 판매자 측 기획 자체가 사라지면서 채울 주체가 없어졌다. 빈 채로 두면 오해를 부른다.

## 판단 근거

- 동선은 흰 테두리(케이싱)를 두껍게 해 교차점이 "다리처럼" 위로 지나가 보이게 하고, 부드러운 그림자로 바닥에서 떠 있는 느낌을 줬다 — 깊이가 곧 우선순위 단서가 된다.
- 줌은 "이미 사용자가 맞춰 둔 화면을 존중"하는 원칙으로, 자동 개입을 최소화(필요할 때만, 약하게)했다.
- 필터는 검색·목록과 같은 맥락(하단 시트)에 모아 손이 가는 곳에 두고, 지도 상단은 비워 전시장을 더 넓게 보여준다.
- 대기 시간은 데이터를 채울 주체가 없는 채로 두느니 깔끔히 걷어내, 보이는 정보는 모두 신뢰할 수 있게 했다. 점수/동선 계산에서도 대기 가중치를 들어내 로직이 단순해졌다.
