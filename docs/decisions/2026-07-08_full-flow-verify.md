## 좋았던 점 (기대대로 나온 것)

- **대화형 온보딩**이 확실히 "폼 입력"에서 "동행자와의 대화"로 인상이 바뀜. 진행 표시(n/3)와
  Roam의 목적·사용법 발화로 "왜 입력하는지"가 해소됨.
- **근거 카드**가 각 부스에 "왜 너에게 맞음 + 확인 가능한 근거(굿즈/팁) + 신뢰도"를 붙여,
  단순 추천이 아니라 사용자가 판단할 재료를 줌. 저작된 부스(16/79)는 가치별 추천 근거가
  그대로 떠서 카드가 눈에 띄게 풍부해짐(예: A1102 "오디오북 체험·라디오 포토존처럼 직접
  해볼 거리가 많아", 근거 충분).
- **리듬(집중/가볍게/쉬면서)**이 피드 밀도를 실제로 바꿈(6/5/4) — 관람을 자기 페이스로.
- **회고 → 재방문 기억**이 이어지며 "이 앱이 나를 기억한다"는 감각이 실제로 성립.
- 전 구간 **런타임/콘솔 에러 0**, 로고 SVG·영상(walking/head/walk_see) 정상.

## 발견된 이슈

### 고침(이번 커밋)
1. **지도 코치마크가 제거된 기능을 안내** — "AI 추천 = 동선을 새로 받아요", "부스 색: 보라=내
   동선"(동선은 Phase A에서 제거). 매 지도 첫 진입마다 노출되는 문구라 영향 큼 → AI추천 항목
   삭제, 색 안내를 "초록=가봄, 노랑=끌린 곳, 반응할수록 관심 진해져"로, 전체 반말화.
2. **로그인/404 잔여 "동선" 카피** — 로그인 시트("저장·메모·동선 공유"), 로그인 폼("나만의
   동선으로"), 404("부스나 동선이 삭제") → 동선 제거 + 컴패니언 톤으로 교체.

### 문서화(후속 — 이번 범위 밖, 별도 정리)
3. **랜드스케이프 전시 페이지 미마이그레이션** — `src/app/landscape/exhibitions/[slug]/page.tsx`가
