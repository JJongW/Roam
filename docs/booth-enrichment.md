# 부스 추가정보 입력 (굿즈 · 테마 · 팁)

운영자/사용자가 부스별 추가 정보를 여기에 채워 넣는 입력 파일이다.
인스타·웹에 있는 내용을 **사람이 보고 옮겨 적는다** (자동 스크래핑 아님).

채운 뒤 처리 흐름:
1. 이 파일을 `booth-data-entry`로 읽음
2. Gemini가 텍스트 → 구조화 태그(굿즈/테마)로 1회 추출
3. `booth_enrichment` 테이블에 저장
4. 추천/온보딩은 저장된 태그를 **LLM 호출 없이 즉시** 읽어 씀
   (Gemini는 추천 때마다 부스를 다시 읽지 않는다 → 빠름)

## 작성 규칙
- 부스 **코드**(A101, B704 …)가 자연키. 코드는 floorplan-sibf.json 기준.
- 빈 칸은 비워도 됨. 아는 것만 채워도 추천이 좋아진다.
- 굿즈/테마는 쉼표로 여러 개. 한국어 자연어 OK (추출 단계에서 정규화).
- 한 부스 = 한 블록. 블록 사이는 `---`.

## 입력 양식 (복사해서 채우기)

```
## [코드] 부스명
- 인스타: 
- 웹: 
- 굿즈: 
- 테마/키워드: 
- 현장 이벤트/혜택: 
- 팁(대기·위치·추천 시간대 등): 
- 자유 메모: 
```

---
