# 지도 AI 추천 채팅바 + Gemini 모델 갱신(전반 안정화)

**날짜**: 2026-06-25
**관련 파일**: map/ai-recommend-sheet.tsx, map-view.tsx, api/ai/quick-route, ai/gemini.ts

## What
- 지도 상단의 "추천"이 "AI 추천받기"로 바뀌고, 누르면 자연어 채팅창이 뜬다(온보딩이 아니라). 적은 내용 + 이미 담은 부스 + 관심사를 함께 고려해 동선을 지도에 바로 그려준다.
- AI 기능 전반(부스 요약·커뮤니티 요약·키워드·추천·재계획)이 안정적으로 동작한다.

## Why
- 별 아이콘 "추천"만으론 뭔지 모호했고, 빠르게 한 줄로 추천받는 경로가 필요했다.
- AI가 자주 실패했는데, 원인은 폴백 모델 gemini-2.0-flash가 단종(404)되어 1차 모델이 잠깐 흔들리면 전체가 실패하던 것.

## 판단 근거
- 채팅 추천은 기존 동선(cart)과 온보딩 관심사를 합쳐 한 번에 재구성(buildOrderedRoute)해, "이미 담은 건 살리고 새로 끼워 넣는" 자연스러운 흐름을 만든다.
- 폴백 모델을 현행 gemini-2.5-flash-lite로 교체 → 1차(2.5-flash) 실패 시에도 살아있는 모델로 폴백되어 AI 호출 성공률이 정상화.
